JCSlider

What is JCSlider?

JCSlider is a responsive, customizable, clean, lightweight, and a very easy-to-use image slider.

Features

Responsive

JCSlider was written to accomodate browsers of different shapes and sizes.

Fully-loaded

JCSlider comes with everything it needs to operate. The only things JCSlider needs from you are the images you want to present and the options you wish to modify or override.

Customizable

You want to put in images that are different sizes? NO PROBLEM.

You don't want the side buttons? DONE.

You only want the slider to be no wider than 650px? SHRINK.

The navigation buttons at the bottom are too distracting? GONE.

You can modify JCSlider's options for every slider or on a per-instance basis. Please visit the wiki page for full details.

The slider above was instantiated and customized by with the following code snippet (imageTwo array omitted): 

var	containerTwo = $('div#sliderTwo'),
	optionsTwo = {
		'width': '650px',
		'sideButtons': 'none',
		'navButtons': false,
		'frameBG': 'rgba(0,0,0,0)',
		'marginTop': '1em'
	},

	sliderTwo = $.jcSlider(containerTwo, imagesTwo, optionsTwo);
